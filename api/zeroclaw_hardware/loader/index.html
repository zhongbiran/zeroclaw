Module loader 

Source
Expand description

Plugin manifest loader — scans ~/.zeroclaw/tools/ at startup.

Layout expected on disk:

~/.zeroclaw/tools/
├── i2c_scan/
│   ├── tool.toml
│   └── i2c_scan.py
└── pwm_set/
    ├── tool.toml
    └── pwm_set

Rules:

  • The directory is created if it does not exist.
  • Each subdirectory is scanned for a tool.toml.
  • Manifests that fail to parse or validate are skipped with a warning; they must not crash startup.
  • Non-directory entries at the top level are silently ignored.
