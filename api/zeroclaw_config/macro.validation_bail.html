Expand description

Per-bail-site shorthand for emitting a structured ConfigApiError inside a validate() chain that returns anyhow::Result<()>. Wraps the structured error as the anyhow source so ConfigApiError::from_validation downcasts to it without having to re-classify the message text. Pattern:

ⓘ
validation_bail!(
    RequiredFieldEmpty,
    "gateway.host",
    "gateway.host must not be empty",
);

Sites not yet converted still bail through anyhow::bail! — the classifier in from_validation covers them as fallback. Migration is incremental: each PR that touches a validate() site swaps the macro in.
