Peripheral trait — hardware boards (STM32, RPi GPIO) that expose tools.

Peripherals are the agent’s “arms and legs”: remote devices that run minimal firmware and expose capabilities (GPIO, sensors, actuators) as tools. See docs/hardware-peripherals-design.md for the communication protocol and firmware integration guide.
