AWS Bedrock provider using the Converse API.

Authentication: supports three methods:

  • Bearer token: set BEDROCK_API_KEY env var (takes precedence).
  • SigV4 signing: AWS AKSK (Access Key ID + Secret Access Key) via environment variables, credential_process in ~/.aws/config, or EC2 IMDSv2. SigV4 signing is implemented manually using hmac/sha2 crates — no AWS SDK dependency.
