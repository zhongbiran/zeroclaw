ProviderActivation
How a provider’s “configured / active” state is computed against a user’s ProvidersConfig. Each ProviderInfo row picks the variant that matches its activation rule, so consumers (the integrations registry, dashboards, doctor) dispatch generically without knowing any provider names. Adding a new provider means adding one ProviderInfo row — no edits at the consumer site.
