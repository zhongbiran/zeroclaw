is_tool_schema_error
Check if an error is a tool schema validation failure (e.g. Groq returning “tool call validation failed: attempted to call tool ‘…’ which was not in request”). These errors should NOT be classified as non-retryable because the provider’s built-in fallback logic (compatible.rs::is_native_tool_schema_unsupported) can recover by switching to prompt-guided tool instructions.
