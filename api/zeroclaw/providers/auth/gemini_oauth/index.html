Module gemini_oauth

Expand description

Google/Gemini OAuth2 authentication flow.

Supports:

  • Authorization code flow with PKCE (loopback redirect)
  • Device code flow for headless environments

Uses the same client credentials as Gemini CLI for compatibility.
