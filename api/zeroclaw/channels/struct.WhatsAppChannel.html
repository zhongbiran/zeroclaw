pub fn apply_mention_gating( dm_patterns: &[Regex], group_patterns: &[Regex], content: &str, is_group: bool, ) -> Option<String>

Apply mention-pattern gating for a message.

Selects the appropriate pattern set based on is_group and applies mention gating: when patterns are non-empty, messages that do not match any pattern are dropped (None); messages that match have the matched fragments stripped. When the applicable pattern set is empty the original content is returned unchanged.
