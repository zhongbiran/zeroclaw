Ed25519 plugin signature verification.

Uses ring (already a dependency) for Ed25519 signing and verification. Plugin manifests may include a base64url-encoded Ed25519 signature over the canonical manifest bytes (TOML content without the signature field). Publisher public keys are stored in the config as hex-encoded strings.
