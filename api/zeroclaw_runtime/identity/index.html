Identity system supporting OpenClaw (markdown) and AIEOS (JSON) formats.

AIEOS (AI Entity Object Specification) is a standardization framework for portable AI identity. This module handles loading and converting AIEOS v1.1 JSON to ZeroClaw’s system prompt format.
