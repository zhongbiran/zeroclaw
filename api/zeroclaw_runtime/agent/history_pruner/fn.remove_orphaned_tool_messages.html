pub fn remove_orphaned_tool_messages(messages: &mut Vec<ChatMessage>) -> usize
Expand description

Remove tool-role messages whose tool_call_id has no matching tool_use / tool_calls entry in a preceding assistant message.

After any history truncation (drain, remove, prune) the first surviving message(s) may be tool results whose assistant request was trimmed away. The Anthropic API (and others) reject these with a 400 error.

Returns the number of messages removed.
