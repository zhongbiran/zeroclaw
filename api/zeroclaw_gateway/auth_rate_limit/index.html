Module auth_rate_limit 

Source
Expand description

Sliding-window rate limiter for authentication attempts.

Protects pairing and bearer-token validation endpoints against brute-force attacks. Tracks per-IP attempt timestamps and enforces a lockout period after too many failures within the sliding window.
